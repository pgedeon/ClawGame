# ClawGame Roadmap

## Current Phase: Milestone 0 (Foundation)

**Goal:** Create repo, package layout, docs skeleton, and initial project metadata model.

### Status: In Progress

## Milestone Timeline

| Milestone | Goal | Status |
|-----------|------|--------|
| **M0: Foundation** | Repo scaffold, docs, metadata model | 🔄 In Progress |
| **M1: Editor Shell** | Dashboard, navigation, project model | ⏳ Not Started |
| **M2: Code + AI** | File tree, code editor, AI commands | ⏳ Not Started |
| **M3: Runtime + Preview** | 2D engine, browser playtest | ⏳ Not Started |
| **M4: Scene Editor** | Visual 2D scene editing | ⏳ Not Started |
| **M5: Asset Pipeline** | ComfyUI integration | ⏳ Not Started |
| **M6: Git + OpenClaw** | Version control, agent-native ops | ⏳ Not Started |

## MVP Acceptance Criteria

1. User can create a project from browser
2. User can describe a feature and AI implements it
3. User can inspect AI-made file changes
4. User can run game in browser preview
5. User can visually edit a 2D scene
6. User can generate an asset via ComfyUI
7. User can init and commit to Git
8. Project has stable metadata for OpenClaw
9. Sample game template can be created and modified
