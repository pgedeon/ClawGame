# ClawGame Vision

## What We're Building

ClawGame is an **AI-first web game engine and editor** that lets humans direct game creation while AI handles implementation, generation, debugging, and iteration.

## Core Belief

**Intent before implementation.**

Users should describe what they want. The system should translate intent into code, assets, tasks, and builds.

## Differentiator

ClawGame is not just a browser game editor with a chatbot attached. It is:

- **AI-first**: AI is the primary mechanism for creating and modifying the game
- **Task-driven**: Work is organized as discrete, trackable tasks
- **Agent-native**: Multiple AI roles (builder, art, director) coordinate
- **Git-native**: Every change is versioned and reviewable
- **OpenClaw-native**: Agents can operate the entire project autonomously

## MVP Goal

Prove that a user can go from idea to playable 2D browser game prototype with meaningful AI assistance inside one integrated system.

## Long-Term Vision

Become a full AI-native creation platform for indie game teams, expanding from 2D into 2.5D and 3D, with reusable templates, workflows, and plugin ecosystems.
